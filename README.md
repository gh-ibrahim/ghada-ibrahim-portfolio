# Ghada Ibrahim - Personal Landing Page

A modern, professional German-style landing page showcasing expertise in digitalization, human-centered design, and international IT program management.

## Overview

This landing page features a sophisticated editorial-inspired design with:
- **Distinctive Typography**: Crimson Pro (serif) for headings and DM Sans for body text
- **Warm Color Palette**: Earthy tones with teal accent colors
- **Professional Aesthetic**: Clean, modern design inspired by Linear and Vercel
- **Fully Responsive**: Optimized for all devices

## Design System

### Colors
- **Background**: Warm cream (#F9F7F4)
- **Foreground**: Dark brown (#1F1611)
- **Accent**: Teal/turquoise (#3A9B9B)
- **Primary**: Rich brown (#2D1A10)
- **Secondary**: Light warm gray

### Typography
- **Headings**: Crimson Pro (serif) - distinctive, elegant
- **Body**: DM Sans (sans-serif) - clean, readable
- **Code/Tags**: JetBrains Mono (monospace)

### Animations
- Fade-in effects on scroll
- Scale-in animations for images
- Hover transitions on interactive elements
- Smooth scrolling between sections

## Technical Stack

- **Framework**: React with TypeScript
- **Styling**: TailwindCSS with custom design system
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Build Tool**: Vite

## Customization

To update content:
1. **Profile Photo**: Replace `/public/ghada-profile.png`
2. **Contact Info**: Update email links in `Contact.tsx` and `Footer.tsx`
3. **Social Links**: Add LinkedIn/GitHub URLs in `Contact.tsx` and `Footer.tsx`
4. **Experience**: Modify arrays in `Projects.tsx`
5. **About Content**: Edit text in `About.tsx`

## Development

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build
```
